# 언맥: 잇는 자 — 대회판 기준 사양

## 장르
2D 사이드뷰 시네마틱 턴제 덱빌딩 로그라이트

## 기술
- React: 손패, 언령 슬롯, HUD, 보상·AI 화면
- TypeScript: 전투 규칙과 콘텐츠 데이터
- Phaser: 픽셀 전투 장면, 배우, 효과, 카메라, 사운드
- Vite: 개발 서버와 웹 빌드

## 렌더 방향
- 내부 전투 해상도: 640×360
- 출력 기준: 1280×720, 정수 배율 확대
- Phaser 캔버스: 전체 화면 뒤
- React UI: 전체 화면 위 투명 오버레이
- 긴 한글 설명은 DOM 텍스트 유지

## 미술
시네마틱 픽셀 아트. 특정 작품의 그림체를 복제하지 않고, 옥빛 언령·탁본·먹 번짐·돌과 종이의 질감을 고유 시각 언어로 사용한다.

## 이번 기술 검증
1. React 기술 카드 → Phaser 적 몸 드래그
2. 공격·피격·틈 연출
3. React 언령 → Phaser 적 행동 드래그
4. 멎는다 발동과 행동 취소 연출
5. 반응형 화면에서 앵커 좌표 유지
