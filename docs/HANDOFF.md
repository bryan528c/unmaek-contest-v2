## 직접 검증할 항목
- 피해를 받았을 때 최근 피해량이 정확히 기록되는가
- 체력바 붉은 잔상이 실제 최근 피해량과 일치하는가
- `되돌린다`가 최근 변화 대상에만 스냅되는가
- 잘못된 대상에서는 언령이 차감되지 않는가
- 발동 시 최근 피해량만큼 체력이 복구되는가
- 최대 체력을 초과하지 않는가
- 언령 자원이 정확히 1회만 차감되는가
- 같은 피해를 두 번 복구할 수 없는가
- 역재생 연출 중 추가 입력이 차단되는가
- 초기화 후 최근 피해 기록과 잔상이 제거되는가
